## 🚀 How to Enable GitHub Pages (for dummies)

1. Go to your repository on GitHub: **github.com/xfalseidol/self**
2. Click the **Settings** tab (top of the page, gear icon)
3. In the left sidebar, scroll down and click **Pages**
4. Under **"Branch"**, open the dropdown and select **`main`**
5. Make sure the folder is set to **`/ (root)`**
6. Click **Save**
7. Wait about 1–2 minutes, then visit `https://xfalseidol.github.io/self/`

That's it. GitHub will host the `index.html` file automatically.

---

## 📄 What the Page Does

- Presents a psychological assessment about relationships and emotional intimacy
- After **5 seconds**, the page switches to a dark "unstable mode" (smooth color transition)
- You can type a critique in the text box and click **Submit Critique**
- The submit button returns a randomly selected "dead to history" error message
